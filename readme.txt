Thank you for using my app!!

What kind of app?
This application reads the latest tweets from your Twitter account 
and displays them on your browser.

Requirements
node.js v14.xx.x(my execution environment -- v14.15.5)
npm  6.xx.xx(my execution enviroment -- v6.14.11)

How to use
1.Enter the Twitter API consumer key,
 access token key, and account name in renew.js.

2.Run "cd dstweet_ver0.0.1" on terminal.

3.Run "npm i" on terminal.

4.Run "node renew.js" on terminal.

5.Run "npm start" on the terminal.
